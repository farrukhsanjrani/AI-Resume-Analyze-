# 🚀 AI Resume Analyzer

AI Resume Analyzer uses Google Gemini AI to audit resumes in PDF/DOCX formats against industry standards. It delivers instant, data-driven optimization reports, ATS scoring, and skill gap analysis via a modern, web-based interface.

---

## ✨ Features

* **Multi-Format Parsing:** Extracts text cleanly from both `.pdf` and `.docx` files using `PyPDF2` and `python-docx`.
* **AI-Powered ATS Scoring:** Generates a realistic numeric benchmark score out of 100 via the Gemini API.
* **Skill & Gap Analysis:** Identifies core professional strengths and highlights missing high-demand industry skills.
* **Smart Career Matches:** Suggests top job roles tailored to the candidate's profile experience.
* **Premium SaaS UI:** Features a sleek dark-mode interface built with Tailwind CSS and FontAwesome icons.
* **Admin Analytics Dashboard:** Includes live interactive graphs powered by Chart.js to track system traffic.

---

## 🛠️ Tech Stack

* **Backend:** Python / Flask
* **AI Intelligence:** Google Gemini API (`gemini-2.5-flash`)
* **File Parsers:** PyPDF2 & python-docx
* **Frontend:** HTML5, Tailwind CSS, Chart.js
